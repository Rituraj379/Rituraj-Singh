# Rituraj Singh Portfolio

A modern, responsive portfolio built with **React + Vite** to showcase my projects, skills, resume highlights, and contact information.

## Live Focus
- Open for **Internship** and **FTE** opportunities
- Strong focus on **Full-Stack Development**, **Systems**, and **Data Science / ML / DL / NLP**

## Tech Stack
- React 19
- Vite 8
- CSS (custom, responsive)
- JavaScript (ES modules)

## Features
- Sticky navigation with active section highlighting
- Smooth reveal-on-scroll animations
- Hero section with profile image
- Optimized **What I Do** section
- Dedicated **Skills** section
- Project showcase cards with GitHub/live links
- Resume timeline (education + achievements)
- Gmail compose integration for contact links
- Mobile-first responsive layout

## Getting Started

### 1) Install dependencies
```bash
npm install
```

### 2) Run development server
```bash
npm run dev
```

### 3) Build for production
```bash
npm run build
```

### 4) Preview production build
```bash
npm run preview
```

## Customization

### Update profile photo
Replace:
- `public/p1.jpeg`

### Update resume file
Replace:
- `public/RiturajSingh_Resume(F).pdf`

### Update content
Edit:
- `src/App.jsx` for text, links, and sections
- `src/styles.css` for design/styling
